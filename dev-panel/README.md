# PAM Dev Panel

A simple web-based control panel for common development tasks.

## Setup

1. Install dependencies:
```bash
cd dev-panel
npm install
```

2. Start the panel:
```bash
npm start
```

3. Open in your browser:
```
http://localhost:3030
```

## Features

- **Git Pull**: Update your code from the repository
- **Git Status**: Check current git status
- **NPM Install**: Install/update dependencies
- **Start Dev Server**: Start the Next.js dev server on port 4000
- **Stop Dev Server**: Stop the running dev server
- **Build**: Build the production version
- **Lint**: Run ESLint checks

## Quick Links

The panel includes quick links to:
- Main app (http://localhost:4000)

- Dashboard

## Status Indicator

The green/gray dot shows if the dev server is currently running.
